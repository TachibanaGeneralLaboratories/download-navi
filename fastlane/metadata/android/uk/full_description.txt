# Особливості

* Безкоштовне програмне забезпечення з відкритим кодом (FOSS)
* Material Design
* Декілька тем (світла, темна, чорна)
* Підтримка Android TV
* Підтримка Chrome OS
Пауза та продовження завантажень
* Підтримка браузерів
* Вбудований браузер
* Одночасні завантаження (початково 3)
* Завантаження декількома частинами (максимум 16)
* Завантаження файлів у фоновому режимі
* Опція «Лише безлімітні з'єднання» (наприклад Wi-Fi)
* Керування роумінгом
* Контроль User Agent
* Обчислення хешу (MD5 та SHA-256)
* Керування живленням і контроль заряду акумулятора
* Можливість призначити різноманітні автоматичні дії після завершення завантаження

# Features

* Software libre y de código abierto (FOSS)
* Material Design
* Múltiples temas (Claro, Oscuro, Negro)
* Soporte para Android TV
* Soporte para Chrome OS
* Pausar y reanudar descargas
* Soporte para navegadores
* Navegador incorporado
* Descargas simultáneas (predeterminado 3)
* Descarga de varias partes (máx. 16)
* Descarga de archivos en segundo plano
* Solo se admiten conexiones sin límite (ej. Wi-Fi)
* Controles de roaming
* Controles del agente de usuario
* Calcular valor hash (MD5 y SHA-256)
* Administración de energía y control de batería
* Varias acciones automáticas después de finalizar la descarga
* Límite de velocidad

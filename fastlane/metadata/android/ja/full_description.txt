# Features

* 無料でオープンソースなソフトウェア (FOSS)
* マテリアルデザイン
* マルチテーマ (ライト, ダーク, ブラック)
* Android TV 対応
* Chrome OS 対応
* ダウンロードの 一時停止/再開
* ブラウザ対応
* 内蔵ブラウザ
* 同時ダウンロード (既定では3個)
* 分割ダウンロード (最大16分割)
* バックグラウンドでのファイルダウンロード
* 定額制接続のみ対応 (例.
Wi-Fi)
* ローミング制御
* ユーザーエージェント制御
* ハッシュ値計算 (MD5 & SHA-256)
* 電源制御とバッテリー管理
* ダウンロード後に様々な動作設定可能
* 速度制限
